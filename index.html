MLB Betting Assistant

Live ↗

A full-stack MLB game prediction tool with a LangGraph-powered AI chatbot, FastAPI backend, and Streamlit UI. Ingests live odds, lineups, weather, and park data to generate over/under predictions using a trained XGBoost model — anchored to Vegas closing lines with adjustments for pitching form, bullpen quality, lineup strength, and game-day conditions.

The ML model uses a residual prediction approach trained on games across four MLB seasons. The chatbot layer lets users ask natural language questions about games, lineups, and predictions through a multi-agent LangGraph workflow.

Music Manager

Live ↗

A full-stack web app that connects to the Spotify API to give you superpowers over your playlists. Four core tools — Discovery is open to all; Shuffler, Updater, and Creator require Spotify account whitelisting.

  • Shuffler — truly randomizes a playlist using Fisher-Yates, replacing the order directly in your Spotify library
  • Updater — refreshes every track by swapping each one for a random song from the same artist
  • Creator — generates a new playlist of any length from a list of artists, picking songs at random from their discographies
  • Discovery — AI-powered recommendation engine using anchor tracks as a taste fingerprint, ranked by cosine similarity with upvote/downvote steering
